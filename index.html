# Volatility change:

Real changepoints: 100,170,240,310,380.
Original data and inferred run length distribution:

Definition:
  • rt is the time lenght since last changepoint, x(r)t denote the data sets associated with run rt. As r may be zero, x(r) may be empty.
  • For each parition ρ, the data within it are i.i.d. from some probability distribution P(xt|ηρ)
  • The discrete a priori(Latin phrase, means ‘from the earlier’,contrast to a posteriori,’from the latter’) probability distribution over the interval between changepoints is denoted as Pgap(g), i.e. the probability of current run ending up at length g.(recall geometric distribution)
  • Predictive distribution P(xt+1|rt,x(r)t)
    Posterior distribution P(rt|x1:t)
    Joint distribution P(rt,x1:t)
Basic Principle:

With the above definitions, our goal here is to recursively(use recusive methods for the purpose of online calculation) compute the probability distribution of the length of the current “run”, or time since the last changepoint, denote this posterior distribution as P(rt|x1:t)
According to bayesian therom,

P(rt|x1:t)=P(rt,x1:t)P(x1:t),(1)

we can devide the calculation of the conditional distribution into two subproblems:
1. Calculate the joint distributioin P(rt,x1:t).
2. Calculate the normalizing constant P(x1:t).
As for subproblem 2, P(x1:t) can be easily calculated by integrate out rt in subproblem 1. i.e. P(x1:t)=∑rtP(rt,x1:t). So basically we need only focus on subproblem 1, the derivation of P(rt,x1:t).
Keep in mind that to make the online calculation possible, the derived represntation of subproblem 1 must be recursive. i.e. when the previous information is known, one need to derive a relation between previous information and current estimation. Here in subproblem 1, the previous information is P(rt−1|x1:t−1), and the current estimation is P(rt|x1:t). In order to relate these two distributions, first we need to disintegrateP(rt|x1:t) by rt−1, and then use bayes rule to extract P(rt−1|x1:t−1):

P(rt|x1:t)====∑rt−1P(rt,rt−1,x1:t)∑rt−1P(rt,xt|rt−1,x1:t−1)P(rt−1,x1:t−1)∑rt−1P(xt|rt,rt−1,x1:t−1)P(rt|rt−1,x1:t−1)P(rt−1,x1:t−1)∑rt−1P(xt|x(r)t)P(rt|rt−1)P(rt−1,x1:t−1)(2)

Remark:

  • By definition, x(r)t⊂x1:t−1. Or more precisely, x(r)t is the last rt(may be zero) elements of x1:t−1.
  • P(xt|x1:t−1,rt or rt−1) relies only on x(r)t, i.e. P(xt|x1:t−1,rt or rt−1)=P(xt|x(r)t). Intuitively, if xt and xt−1 belong to the same partition, then the prediction probability of xt is P(xt|ηt−1), where ηt−1 is determined by x(r)t, so we have P(xt|ηt−1)=P(xt|x(r)t). On the other hand, if xt belongs to a new partition, then xt is irrevelant to ηt−1, the equation still holds: P(xt)=P(xt|ηt−1)=P(xt|x(r)t)
  • rt only depends on rt−1.

According to (2), subproblem 1 is further devided into 3 minus subproblems:
(a). Construct a boundary condition, or the initial distribution of P(rt−1,x1:t−1). When t=1, P(rt−1,x1:t−1)=P(r0,x0), note that x0 is an empty set, so the problem can be simplified to the construction of P(r0).
(b). Construct a transition distribution P(rt|rt−1)
(c). Construct a prediction distribution P(xt|x(r)t)
As to (a), the simplist way is to assume a changepoint occured before the first data. In such cases we place all of the probability mass for the initial run length at zero, i.e. P(r0=0)=1.
As to (b), P(rt|rt−1)has non-zero mass at only two outcomes: the run length either continues to grow and rt=rt−1+1 or a changepoint occurs and rt=0. This reasoning process is the same as the one used in inferring motality rate, which can be easily interpretated by a hazard function(hazard=densitysurvival). Here we have:

P(rt|rt−1)=H(rt−1+1),1−H(rt−1+1,0,if rt=0,if rt=rt−1+1other.(3)

Where

H(g=τ)=Pgap(g=τ)∑infinityi=τPgap(g=i).(4)

Any prior information about run length transitions can be easily incoporated by the form of Pgap(g). For example, if Pgap(g) is an exponential(or geometric, if discrete) distribution with timesacle λ, then the hazard function is constant(independent of previous run length), H(g)=1/λ. It is the same as making no prior assumptions on run length transitions.
As to (c). Recall that in bayesian point of view, if every data point x belongs to a distribution P(x|η), and η belongs to a prior distribution P(η|θ), where θ is the hyperparameters. Then the posterior distribution of η can be shown satisfying P(η|x)=P(x|η)P(η|θ)/P(x). Here in this problem. let ηt be the parameter inferred from x(r)t, then we have:

P(ηt|x(r)t)=P(x(r)t|ηt)P(ηt|θ)/P(x1:t−1).(5)

And the prediction distribution can be easily calculated when disintegrated by ηt(bayesian prediction problem):

P(xt|x(r)t)=∫ηtP(xt|ηt)P(ηt|x(r)t)=∫ηtP(xt|ηt)(5)(6)

So far we have only one problem left, i.e. the derivation of (5). There are three parts in (5), they are the likelihood functionP(x(r)t|ηt)(or the sample distribution P(x|ηt), the two are technically implying the same thing), the prior distributionP(ηt|θ) and the normalizing constant P(x1:t−1). Because the normalizing constant is already dealt in subproblem 2, here we only need to find a proper likelihood function and a prior distribution.
Generally, we need to calculate the posterior distribution of ηt directly from (5), but sometimes it would be vary time consuming. Particularly, if both likelihood and prior distribution are from exponential family, the posterior distribution will also from exponential family, and allow inference with a finite number of sufficient statistics which can be calculated incrementally as data arrives.
Here is a general case where there is a normal ikelihood function with unkonwn mean μ and precision τ(or inverse of variance,1/σ2). Usually, if τ is known, μ follows a noramal distribution with mean μ0 and precision k0, and if μ is known, τ follows a gamma distribution with shape α0 and rate(1/sclale) β0. When both μ and τ are unknown, it is usual to assme they follow a normal-gamma distribution with parameters μ0,k0,α0 and β0 :

NG(μ,τ)=N(μ|(k0τ)−1)Ga(τ|α0,β0)

see exponettial family and Conjugate Bayesian analysis of the Gaussian distribution for more information.
Note

  1. In the above example c(k0,α0) and c(μ0,β0) act the same as effective number of observations v and the amount contribute to the sufficient statistic χ repestectively.
  2. In the bayesian prediction problem (6), calculate the integration ∫ηp(xt|η)p(η|x1:t−1) directly will be too time consuming. Alternatively, we can use the expected value of η instead of the whole distribution to approximate the probability. i.e.

    ∫ηp(xt|η)p(η|x1:t−1)≈p(xt|E(η|x1:t−1))
